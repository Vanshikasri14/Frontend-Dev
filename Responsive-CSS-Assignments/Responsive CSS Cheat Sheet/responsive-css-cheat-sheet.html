Responsive CSS Cheat Sheet

Filters

filter: grayscale(100%); applies grayscale to images or elements.

Media Queries

@media (max-width: 700px) { ... } switches styles for small screens.

Responsive Design

Design that adapts to device size using flexible layouts and media queries.

Typography

font-size: clamp(1rem, 2vw, 1.5rem); scales type smoothly.
