Fluid Layouts

Use relative units and flexible grids to let content flow.

grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));

Flexible Media

Constrain images/videos to container width while keeping aspect.

img { max-width: 100%; height: auto; }

Breakpoints

Use media queries to adjust layout at key widths.

@media (max-width: 600px) { ... }
