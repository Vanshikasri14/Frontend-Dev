Frequently Asked Questions

Click a question to toggle the answer. Double-click a question to collapse all answers.

1. What is jQuery?

jQuery is a fast, small, and feature-rich JavaScript library.

2. Why use jQuery?

It simplifies HTML document traversal, event handling, and animations.

3. What is a selector?

A selector is a pattern used to select DOM elements.

4. What is a callback function?

A function passed into another function as an argument to be executed later.

5. How do I include jQuery?

You can include it via CDN using a <script> tag in your HTML.
